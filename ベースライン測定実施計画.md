# ベースライン測定実施計画
**イカリ消毒株式会社様 DXコンサルティングPoC**

**作成日**: 2025年11月17日
**作成者**: 株式会社ミオシステム
**目的**: DX推進の起点となる現状値を正確に測定し、改善効果を定量的に評価できる基盤を構築する

---

## 📋 1. ベースライン測定の全体像

### 1.1 測定の目的

1. **現状の可視化**: 数値で客観的に現状を把握
2. **改善余地の特定**: どこに改善機会があるかを明確化
3. **目標設定の根拠**: 実現可能な目標値の設定
4. **効果測定の基準**: Before/After比較の起点

### 1.2 測定スコープ

| 測定カテゴリ | 測定項目数 | 対象部門 | 測定期間 |
|------------|-----------|---------|---------|
| 業務効率指標 | 25項目 | 全部門 | 2週間 |
| システム利用指標 | 15項目 | 全部門 | 1週間 |
| データ管理指標 | 10項目 | 情報システムG、各部門 | 1週間 |
| 組織・人材指標 | 8項目 | 人事部、全従業員 | 2週間 |
| 顧客指標 | 5項目 | 営業部門 | 1週間 |
| **合計** | **63項目** | **全社** | **2〜3週間** |

## 🛠️ 3. 測定ツール・テンプレート

### 3.1 業務時間記録シート（Excel）

**ファイル名**: `業務時間記録シート_部門名_氏名.xlsx`

#### シート構成

| 列 | 項目 | 記入内容 |
|----|------|---------|
| A | 日付 | YYYY/MM/DD |
| B | 開始時刻 | HH:MM |
| C | 終了時刻 | HH:MM |
| D | 業務分類 | プルダウン選択（見積作成/商談/移動/日報作成/等） |
| E | 業務内容詳細 | 自由記述 |
| F | 件数 | 処理件数 |
| G | ツール/システム | 使用したシステム・ツール |
| H | 手作業/システム | プルダウン（手作業/システム/半自動） |
| I | 備考 | 気づき・課題等 |

**記入例**:
```
2025/11/18 | 09:00 | 10:30 | 見積作成 | A社向け見積書作成 | 3件 | Excel | 手作業 | テンプレート探しに時間がかかった
2025/11/18 | 10:30 | 11:00 | CRM入力 | 商談記録入力 | 2件 | CRMシステム | システム | -
```

#### 自動集計シート
- 業務分類別の総時間
- 1件あたりの平均処理時間
- 手作業比率
- グラフ表示
