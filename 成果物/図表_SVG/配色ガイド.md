## 📊 適用した配色の原則

### 1. 基本ルール
- **3色の使用**: メインカラー、アクセントカラー、テキストの色で構成
- **70:25:5の法則**: 背景色70%、メインカラー25%、アクセントカラー5%
- **原色を避ける**: 彩度を落とした色で目に優しく
- **コントラスト確保**: 背景と文字の明度差を意識

### 2. 階層表現の原則
濃淡の異なる青を使い分けることで、視覚的な階層構造を表現

## 🎨 採用した配色パターン

### メインパレット: 青系モノトーン（信頼感・プロフェッショナル）

| 用途 | カラー名 | カラーコード | 適用箇所 |
|------|---------|--------------|----------|
| **最重要** | 濃青 | `#1A43AA` | 経営層、基幹システム |
| **主要機能** | 明青 | `#15A2E9` | 主要部門、業務システム |
| **特定業務** | ブルーグリーン | `#33BEC5` | 検査システム、技術部門 |
| **サポート** | グレー | `#708090` | 管理部門、外部システム |
| **強調・注意** | オレンジ | `#F47F29` / `#F17C1D` | 委員会、非システム領域、DX推進 |

### 階層別の配色（組織図用）

```
経営層（最上位）     → 濃青 #1A43AA
  ↓
主要部門（中位）     → 明青 #15A2E9
  ↓
サブ部門（下位）     → 薄青 #5ec4f5
  ↓
グループ（最下位）   → 最薄青 #a8dff8
  ↓
管理・サポート       → グレー #708090
委員会・重点領域     → オレンジ #F47F29
```

## ✅ 配色変更のメリット

### Before（変更前）
- 原色（赤・紫・緑・オレンジ）が混在
- 色数が多く視覚的にうるさい
- 階層構造が色で表現されていない
- 強調箇所が不明瞭

### After（変更後）
- **青系モノトーンで統一感** → プロフェッショナルな印象
- **70:25:5の法則に準拠** → 見やすい配色バランス
- **濃淡で階層を表現** → 組織構造が一目で理解可能
- **オレンジアクセントで重点を強調** → DX推進領域・課題箇所が明確

## 📐 技術仕様

### CSSクラス構造
すべてのSVGファイルで以下の構造を採用:

```css
/* テキストスタイル */
.title { fill: #1a1a1a; }  /* タイトル: 黒 */
.subtitle { fill: #666; }   /* サブタイトル: グレー */

/* 推奨配色パターン適用 */
/* 濃青 #1A43AA / 明青 #15A2E9 / ブルーグリーン #33BEC5 / オレンジ #F47F29 / グレー #708090 */

/* 用途別クラス */
.exec-box { fill: #1A43AA; stroke: #0d2459; stroke-width: 2; }
.dept-box { fill: #15A2E9; stroke: #0d7ab8; stroke-width: 2; }
/* ... */
```

### レスポンシブ対応
すべてのSVGで以下の属性を設定済み:

```svg
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 [width] [height]"
     width="100%"
     height="100%"
     preserveAspectRatio="xMidYMid meet">
```

これにより、Gapplinなどのツールで拡大・縮小・画面フィットが可能。

## 🎯 今後の推奨事項

### 1. 新規図表作成時
新しいSVG図表を作成する際は、本ガイドの配色パレットを使用してください。

### 2. PowerPoint資料への展開
本配色パターンはPowerPointスライドにも適用可能です:
- スライドマスターで色設定を一括管理
- RGB値を正確に入力（例: `#1A43AA` → R:26, G:67, B:170）

### 3. 配色の一貫性
すべての成果物（図表、スライド、レポート）で同じ配色を使用することで、**ブランド統一感**と**プロフェッショナルな印象**を醸成できます。
