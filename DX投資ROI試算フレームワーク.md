## 📊 1. ROI試算の基本フレームワーク

### 1.1 ROI計算式

```
ROI (%) = [(累積利益 - 累積投資額) / 累積投資額] × 100

累積利益 = Σ(年間削減コスト + 年間増収効果 - 年間運用コスト)
累積投資額 = 初期投資額 + Σ年間追加投資額
```

### 1.2 投資回収期間（Payback Period）

```
投資回収期間 = 初期投資額 / 年間平均利益
```

### 1.3 正味現在価値（NPV: Net Present Value）

```
NPV = Σ [キャッシュフロー(t) / (1 + 割引率)^t] - 初期投資額

割引率: 5%を想定（企業の資本コスト）
```

## 📈 3. 効果（ベネフィット）の分類と試算方法

### 3.1 直接的コスト削減効果

#### A. 人件費削減

```
年間削減額 = 削減工数(h/年) × 平均時給 × (1 + 間接費率)

削減工数 = 対象業務の年間総工数 × 削減率
平均時給 = 平均年収 / 年間労働時間
間接費率 = 0.3〜0.5（社会保険料、福利厚生費等）
```

**計算例**: 見積作成業務の効率化
```
現状: 1件30分、月間200件 = 100時間/月 = 1,200時間/年
改善後: 1件10分（-67%削減）= 33時間/月 = 400時間/年
削減工数: 800時間/年
平均時給: 400万円 / 2,000時間 = 2,000円/時
間接費率: 0.4
年間削減額 = 800時間 × 2,000円 × 1.4 = 224万円/年
```

#### B. 外注費削減

```
年間削減額 = (外注前年間費用 - 外注後年間費用)
```

#### C. 紙・印刷費削減

```
年間削減額 = 削減枚数 × 単価 + 印刷機保守費削減額
```

**計算例**: ペーパーレス化
```
現状: 年間100万枚（A4）× 5円 = 500万円
ペーパーレス化: 80%削減
年間削減額 = 100万枚 × 80% × 5円 = 400万円/年
```

#### D. システム統合によるコスト削減

```
年間削減額 = (統合前ライセンス費総額 - 統合後ライセンス費)
```

### 3.2 間接的コスト削減効果

#### A. ミス・手戻り削減

```
年間削減額 = 年間ミス発生件数 × 1件あたり手戻りコスト × 削減率
```

**計算例**: 検査報告書の修正・再発行削減
```
現状: 年間50件の修正・再発行
1件あたりコスト: 検査やり直し3時間 + 報告書作成2時間 = 5時間 × 3,000円 = 15,000円
改善: 80%削減（システムチェック機能導入）
年間削減額 = 50件 × 15,000円 × 80% = 60万円/年
```

#### B. 在庫適正化・キャッシュフロー改善

```
年間削減額 = 削減在庫額 × 資本コスト率
```

#### C. 機会損失の削減

```
年間削減額 = (スピードアップによる追加受注額 × 利益率)
```

### 3.3 増収効果

#### A. 営業力強化による売上増

```
年間増収額 = (商談成約率向上 + 顧客単価向上 + 新規顧客増) による増収
年間利益増 = 年間増収額 × 営業利益率
```

**計算例**: CRM導入による営業力強化
```
現状売上: 50億円
商談成約率: 30% → 35% (+5pt)
見込み増収: 50億円 × (35/30 - 1) ≈ 8.3億円
営業利益率: 10%
年間利益増 = 8.3億円 × 10% = 8,300万円/年
```

#### B. 顧客満足度向上による継続率改善

```
年間増収額 = 既存顧客売上 × 継続率向上率
年間利益増 = 年間増収額 × 営業利益率
```

#### C. 新規サービス・事業機会

```
年間利益増 = 新規サービス売上 × 利益率
```

### 3.4 リスク削減効果

#### A. セキュリティインシデント回避

```
期待削減額 = インシデント発生確率 × 想定損失額 × リスク低減率
```

#### B. コンプライアンス違反回避

```
期待削減額 = 違反発生確率 × 罰金・対応コスト × リスク低減率
```

## 🎯 6. ROI最大化のポイント

### 6.1 効果を高める施策

   - システム導入だけでなく、業務プロセスを抜本的に見直す

   - 早期に成果が出る施策を優先
   - 社内の推進機運を高める

   - 初期投資を抑える
   - スモールスタートが可能

   - 開発コストを削減
   - 内製化を推進

3. **オープンソース活用**
   - ライセンス費削減

4. **段階的投資**
   - 一気に投資せず、効果を見ながら拡大

## ✅ 9. 本フレームワークの正しい使用方法

### ステップ1: ベースライン測定の実施
- `ベースライン測定実施計画.md`に従って63項目を実測
- 業務時間記録シート、全社アンケート、システムログ分析を実施
- 3週間で現状値を取得

### ステップ2: 実測値の当てはめ
本フレームワークの計算式に実測値を代入し、正確なROIを算出します。

**例**: 見積作成時間が実測で「1件45分」だった場合
```
現状: 1件45分、月間200件 = 150時間/月 = 1,800時間/年
改善後: 1件15分（-67%削減） = 600時間/年
削減工数: 1,200時間/年
平均時給: [実測値] 円/時（ベースライン測定で取得）
年間削減額 = 1,200時間 × [実測時給] × 1.4 = [正確な算出値]円/年
```

### ステップ3: ベンダー見積取得
- 優先施策について複数ベンダーから正式見積取得
- 初期投資額、年間運用コスト、保守費用を確定
- 相見積により適正価格を確認

### ステップ4: 正確なROI算出
```
ROI = [(実測削減効果 + 実測増収効果 - 運用コスト) - ベンダー見積額] / ベンダー見積額 × 100
```
- 保守的シナリオ（効果-20%）・楽観的シナリオ（効果+20%）の両建て
- 感度分析（変数を変えた場合のROI変動）

### ステップ5: 経営層への提案
エビデンスに基づくROI試算資料を作成し、投資判断を仰ぎます。
