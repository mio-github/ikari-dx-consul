# イカリ消毒様 DXコンサル支援ツール

## 概要

イカリ消毒株式会社様のDXコンサルティングプロジェクトを支援するReactベースの管理ツールです。
プロジェクト進捗、KPI監視、リスク管理、ドキュメント管理を統合的に行えます。

## 主要機能

### 1. ダッシュボード
- プロジェクト全体の進捗状況を一目で把握
- 主要指標（完了タスク、未解決課題、リスク項目）の可視化
- フェーズ別進捗、課題優先度分布、工数推移、成果物完成状況のチャート表示
- 最近のアクティビティ一覧

### 2. プロジェクト進行管理
- マイルストーンベースのタイムライン表示
- 各フェーズのタスク進捗管理
- 成果物の追跡
- 次回予定のミーティング・イベント管理

### 3. KPI監視
- プロジェクト完遂率、業務効率化達成度、満足度などの主要KPI
- DX成熟度のレーダーチャート
- 月次KPI推移
- 部門別DX進捗状況
- ROI（投資対効果）予測
- KPIアラート機能

### 4. リスク管理
- リスクの特定・評価・対策管理
- リスクマトリックス（影響度 × 発生確率）
- 高・中・低リスクの分類と優先度付け
- リスク詳細情報（対策、担当者、期限）

### 5. ドキュメント管理
- 全成果物の一覧表示
- カテゴリ別フィルタリング（レポート、Excel、Word、図表）
- ドキュメントのステータス管理（完成/作成予定）
- フォルダ構成の可視化
- エビデンス・参考資料の管理

## 技術スタック

- **フレームワーク**: React 18
- **ビルドツール**: Vite
- **チャートライブラリ**: Recharts
- **アイコン**: Lucide React
- **スタイリング**: CSS（カスタム）

## セットアップ

### 前提条件

- Node.js 16以上
- npm または yarn

### インストール

```bash
cd dx-console-tool
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 にアクセスしてください。

### ビルド

```bash
npm run build
```

ビルド成果物は `dist/` ディレクトリに生成されます。

### プレビュー

```bash
npm run preview
```

## 機能詳細

### ダッシュボード

**表示内容**:
- 全体進捗率、完了タスク数、未解決課題数、リスク項目数
- フェーズ別進捗バー（事前準備、PoC開始、中間成果、最終成果）
- 課題優先度分布（円グラフ）
- 工数推移（計画vs実績の棒グラフ）
- 成果物完成状況（積み上げ棒グラフ）
- 最近のアクティビティ（テーブル）

**データソース**: モックデータ（実運用時は API 連携を想定）

### KPI監視

**監視指標**:
- プロジェクト完遂率
- 業務効率化達成度
- ステークホルダー満足度
- ドキュメント完成率
- DX成熟度（6カテゴリ）
- 月次KPI推移（完遂率、効率化、満足度）
- 部門別DX進捗
- ROI予測（5年間）

**アラート機能**:
- 部門別進捗のバラツキ検知
- ROI達成予測の通知

### リスク管理

**リスク分類**:
- 技術リスク
- 組織リスク
- 予算リスク
- 人材リスク
- セキュリティリスク
- 業務リスク
- 外部リスク

**評価軸**:
- 影響度（高・中・低）
- 発生確率（高・中・低）
- リスクスコア（1〜9）

**リスクマトリックス**:
影響度と発生確率の2軸でリスクを可視化

### ドキュメント管理

**ドキュメント分類**:
- レポート（業界動向、事例集、技術要件定義書等）
- Excel（ロードマップ、課題管理表、リスク管理表）
- Word（提案書）
- 図表（組織構造図、システム構成図）

**機能**:
- カテゴリ別フィルタリング
- ステータス管理（完成/作成予定）
- ファイル情報表示（サイズ、形式、更新日）
- プレビュー・ダウンロード（モック）
- エビデンス・参考資料の一覧

## カスタマイズ

### データの更新

各コンポーネント内のモックデータを実際のデータに置き換えることで、リアルタイム監視が可能になります。

```javascript
// 例: Dashboard.jsx
const progressData = [
  // この配列を API から取得したデータに置き換え
];
```

### API連携の追加

```javascript
// 例: useEffect でデータ取得
useEffect(() => {
  fetch('/api/progress')
    .then(res => res.json())
    .then(data => setProgressData(data));
}, []);
```

### スタイルのカスタマイズ

`App.css` でグローバルスタイル、各コンポーネントでインラインスタイルを調整できます。

```css
/* App.css */
.app-header {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  /* 色を変更する場合はここを編集 */
}
```
